**GlyphForge** is a modern, open-source tool to visualize, inspect, and export glyphs from font files. It is available as both a web application and a standalone desktop application.

It allows you to load `.ttf` and `.otf` fonts, browse glyphs by Unicode blocks, customize rendering settings, and export selected glyphs as SVG or PNG â€” completely offline.
